## 🎯 מושגי מפתח בקורס

| מושג | הסבר |
|------|------|
| **פסיכולוגיה** | המחקר המדעי של התנהגות ותהליכים מנטליים |
| **ביהביוריזם** | גישה המתמקדת בהתנהגות נצפית בלבד |
